# Buscador de palavras em JAVA

Projeto em Java que conta e destaca ocorrências de uma palvra em um texto, ignorando maiúsculas/minúsculas e preservando o formato original.

# Tecnologias usadas

- Java 25
- Regex (Pattern/Matcher)

## Como executar

1. Compile o projeto
2. Execute a classe Main
3. Cole o texto e informe a palavra (target)

## Objetivo

Projeto de estudo para praticar manipulação de string e regex em Java
